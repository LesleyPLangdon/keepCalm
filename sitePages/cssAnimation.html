2D Transform

Transform Methods:

translate(xDistance, yDistance)
- moves an element on the x or y axis
rotate(deg)
- rotates an element
scale(xDirection, yDirection)
- increases or decreases an element's size in the specified direction
skew(xAngle, yAngle)
- skews an element along the x or y axis
matrix
- takes 6 transform method values. scaleX(), skeyY(), skewX(), scaleY(), translateX(), and translateY()

DOM

What is the DOM?

  • A programming interface that represents HTML files in a way that is readable by other programming languages.
  • A tree of HTML objects.
  • How we access HTML elements to change them with javaScript.
  • A W3C standard fro accessing documents.

Let's use the DOM

Key terms

document.getElementById("")
Method to access the HTML element with the specified id.
document.getElementsByClassName("")
Method that returns an array-like collection of HTML elements with the specified class.
document.querySelector("")
Method that returns the first occurrence of the sepcified selector.
document.querySelectorAll("")
Method that returns an array-like list of all occurrencesthe sepcified selector.
innerHTML
A property of an HTML element

Arrays

Ways to declare arrays

Constructor
- const NUMS = new Array(1, 2, 3, 4, 5)
Array literal
- const NUMS = [1, 2, 3, 4, 5]
Different data types
- const NUMS = [‘one’, true, 3, ‘for’, 5];

Manipulating the array

Access an item by index
- let num1 = NUMS[0];
Add to the end of an array
- NUMS[5] = 'six'
Find the length of an array
- NUMS.length;
Add to the end of an array of unknown length
- NUMS.push('six')
Add to the beginning of an array
- NUMS.unshift(0);
Remove the last item from an array
- NUMS.pop();
Get the index of an array
- NUMS.indexOf(3);
